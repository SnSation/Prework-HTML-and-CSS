Semantic Markup

This is strong text - Tag: 'strong'
'strong': An element which indicates that its content has strong importance.

This is empasized text - Tag: 'em'
'em': An element which indicates the emphasis of its content changes the meaning of the surrounding content.

This paragraph is in a blockquote element - Tag: 'blockquote'
'blockquote': An element used for longer quotes which take up an entire paragraph. Browsers tend to indent the contents of a blockquote

This is quoted text - Tag: 'q'
'q': An element used for shorter quotes which sit within a paragraph.

Blockquotes and quotes can use the 'cite' attribute to indicate the source of the quote. The cite attribute's value should be a URL.

This is text has an abbr in it.
The 'title' attribute is added to the 'abbr' tag with the value of the full term.

This is cited text - Tag: 'cite'

This is defined text - Tag: 'dfn'

This is linked text - Tag: 'a'
The 'href' attribute is used to indicate where the link should go.
