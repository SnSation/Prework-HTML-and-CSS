CSS Syntax

    CSS rule-sets consist of a "selector" and "declaration"
  • 'Selector': The HTML element to be modified
  • 'Declaration': The modification to take place
  • Examples:
    • h1 {color:blue;font-size:12px
    • The selector - 'h1' - indicates this rule-set will apply to 'h1' elements
    • The declaration - {color:blue; font-size:12px} - indicates the color should be blue and the font-size should be 12px
      • There can be multiple declarations in a rule-set, separated by ";"
      • Declarations are key-value pairs
    • #paragraph1 {
           text-align: center;
           color: red;
      }
      • Selectors can be 'id' values of HTML elements
      • Rule-sets can be written across multiple lines
    • p.center {
           text-align; center;
           color:red;
      }
      • Selectos can target all elements of a class with "."
      • The above rule-set targets all <p> with a 'class' attribute value of "center"
    • * {text-align:center;}
      • The "*" symbol targets ALL HTML elements on the page
    • h1, h2, h3, p {text-align:center;}
      • A selector can target multiple elements with the same declarations by separating them with a ","

Adding CSS

'Style Sheet': A set of CSS modifications

    Style Sheets can be added to HTML in 3 ways
  • External CSS
    • A style sheet can be written in a separate file and linked to the HTML document
    • <link>: An element defined in <head> which links external items to an HTML file
    • Example:
      • <link rel="stylesheet" href="CSStutorial.css">
      • 'rel': An attribute of <link> which defines what kind of item is being linked
  • Internal CSS
    • A style sheet can be written directly in the <head> of an HTML document
    • <style>: An element defined in <head> which indicates its contens are a style sheet
    • Example:
      • <head>
        <style>
        body {
        background-color: blue;
        }
        h1 {
        color: red;
        margin-left: 40px;
        }
        <style>
        <head>
  • Inline CSS
    • A style sheet can be written for a specific element as its attributes
    • Example:
      • <p style="color:red;">This paragraph will be red</p>
  • HTML documents can support multiple style sheets at once
  • Style Sheets will be applied in the following order (cascade):
    1. Inline Styles
    2. External and Internal Styles in the order they appear in <head>
    3. Browser Default Styles
