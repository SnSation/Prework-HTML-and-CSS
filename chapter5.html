Chapter 5: Images

    Images Should:
  • Be Relevant
  • Convey Information
  • Be Recognizable
  • Match the Page's Color Palette

Working With Images

Storage

  • Images for a page should be stored in their own directory
  • This directory should be a child of the folder which holds the page's related data

Adding Images

  • '<img />'
    • An element which contains an image
    • <img /> is a self-enclosed element, meaning it does not need a closing tag
    • 'src'
      • An attribute of <img /> where the value is the image file to load
    • 'alt'
      • An attribute of <img /> where the value is placeholder text should the image not load
    • 'title'
      • An attribute of <img /> where the value is the text that shows when the image is moused over
    • Example: <img src="exampleimage.jpg" alt="Example Image" title="Example Title" />

Sizing Images

Images can be forced to fill a space with HTML

  • 'height'
    • An attribute of <img /> which dictates the height of the image
  • 'width'
    • An attribute of <img /> which dictates the width of an image
  • 'height' and 'width' use pixels as units
  • Example: <img src="assets/images/exampleimage.jpg" width="600" height="450" />
    • This will load the image as if it were 600 pixels wide and 450 pixels tall

Placing Images

  • Where in the code the <img /> is placed affects where on the page it will be located
    • Before an Element: The next element will begin on the line after <img />
    • Beginning in an Element: The first line of text will align with the bottom of <img />
    • Within the Content of an Element: <img /> will be placed between the content surrounding it
      • <img /> will force space between content above and content on the same line as <img />
  • Alignment
    • 'align'
      • An attribute of <img /> to indicate how surrounding content should be placed around the <img />
      • 'left': A value of 'align' which indicates for text to flow to the right of the <img />
      • 'right': A value of 'align' which indicates for text to flow to the left of the <img />
      • 'align' is a legacy attribute, but is still found on many pages

Image Rules

  1. Save Images in the Correct Format
  2. Save Images in the Correct Size
  3. Use the the Correct Resolution
    • Resolution: The number of pixels that fit in a given area
    • The standard resolution is 72 ppi
    • Resolution is interpreted in different ways by different systems
      • This can be how many pixels fit in a static area or, inversely, how much space does a static amount of pixels use

Image Formats

There are many filetypes for Images

  • .JPEG: A bitmap filetype good for images with color
  • .PNG:A bitmap filetype good for images with few colors
  • .GIF: A multi-use filetype for static images or video
  • .SVG A vector filetype which maintains resolution at any size

Figure and Caption

  • <figure>: An element used to contain a <img /> with an associated caption
  • <figcaption>: An element containing a caption associated with an image
  • Example:
    <figure>
         <img src="images/exampleimage.jpg" alt="Example Image" />
         <figcaption>Example Caption for Example Image</figcaption>
    </figure>
