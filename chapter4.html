Chapter 4: Links

Links

  • Link:
    • An element which connects one item to another
    • Tag: 'a'
  • Reference:
    • An attribute which indicates what the link should connect to
    • Attribute: 'href'
  • Syntax:
    • <a href="https://www.sample.com">Sample Website</a>
    • <a href="chapter3.html">Chapter 3 File</a>
      • Links to other files require the file path as written in a command line interface
  • Email Links:
    • 'mailto:'
      • A value of 'href' which opens the user's email application with the value as the recipient
      • Example: <a href="mailto:person@sample.com">Email Person<a>
  • 'target:'
    • An attribute of <a> which indicates where to send the linked item
    • Example: <a href="http://www.google.com/" target="_blank">Google</a>
      • This will open the google website in a new browser window
  • 'id:'
    • An attribute which adds an identifier to an element
    • Links can reference 'id' values
    • Example: <h1 id="top">Top of the Page</h1>
      • The 'id' attribute marks this specific <h1> as "top"
      • A link that references the 'id' value "top" will send the user to the part of the page it is located
    • '#' at the beginning of a reference indicates the reference is to an element's 'id'
    • Example: <p><a href="#top">Go to Top</a></p>
      • This link will send a user to the element with an 'id' value of "top" on the same page
    • 'id' can be used to link to a specific part of another page
      • Example: <p><a href="https://www.examplepage.com/#top">Example Page</a></p>
        • This link will go to the element with the 'id' value "top" on the page "https:/www.examplepage.com/"
