Chapter 3: Lists

Ordered List: A list where each item is numbered.

Unordered List: A list where each item begins with a bullet point.

Definition List: A list where each item is a set of both a term and a definition

Ordered Lists

  1. This is an ordered list
  2. Tag: 'ol'
  3. An ordered list contains list items
  4. Tag: 'li'
  5. List items are individual the items in a list
  6. Most browsers indent lists by default
